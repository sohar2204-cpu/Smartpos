🛠 One-time Setup Guide

1

Create a Google Cloud Project

Go to console.cloud.google.com, create a new project (or pick an existing one), then search for and enable the Google Drive API.

2

Create a Service Account

IAM & Admin → Service Accounts → Create Service Account.
Give it any name (e.g. smartpos-backup). No special IAM roles required.

3

Download the JSON Key

Open the service account → Keys tab → Add Key → Create new key → JSON.
A .json file downloads. Open it in any text editor, select all, and paste the entire contents into the credentials box on the right.

4

Share a Drive Folder with the service account

Create a folder in drive.google.com. Right-click → Share → paste the client_email from the JSON (e.g. smartpos-backup@your-project.iam.gserviceaccount.com) and grant Editor access.
Then copy the folder ID from the URL (…/folders/THIS_PART) and paste it below.

5

Install required Python packages (once, on your server)

pip install google-auth google-api-python-client google-auth-httplib2 openpyxl
