📋 Quick Setup Guide
Step 1 — Create Meta App
Go to developers.facebook.com → Create App → Business type → Add WhatsApp product
Step 2 — Get API Credentials
WhatsApp → API Setup → Copy your Phone Number ID and generate a Permanent Token (System User Token recommended)
Step 3 — Create Message Template
WhatsApp → Message Templates → Create template
• Category: Utility
• Name: receipt (or whatever you set above)
• Body: Add one variable {{1}} — this is where the receipt text goes
• Wait for Meta approval (usually 1–24 hours)
Step 4 — Test it
Open any receipt → enter a phone number → click Send via WhatsApp.
The customer will receive a WhatsApp message with the full receipt. ✅
